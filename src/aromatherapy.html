Ароматерапия -

Эффективный метод использования эфирных масел для оздоровления организма, восстановления эмоционального баланса и решения глубинных запросов.

В нашем салоне для проведения аромапроцедур используются только натуральные эфирные масла премиального качества английского бренда M.Nectar, которые добываются максимально чистым способом, без применения химических реагентов. Масла не содержат синтетических отдушек, абсолютно безопасны даже для людей с высокой чувствительностью.

При работе с маслами мы всегда используем принцип глубокого одобрения ароматов – это значит, что каждый гость сам выбирает эфирные масла, которые ему максимально приятны, потому что именно они способны дать наилучший терапевтический эффект.
